Projet BTS SIO — Application desktop JavaFX

GestTravaux Pro - Desktop

Application desktop développée en JavaFX pour les gestionnaires du syndic IMMOSYNC. Interface de pilotage complet avec analyse comparative des devis, visualisation des chantiers et outils de décision avancés.

  • Java / JavaFX
  • PostgreSQL
  • Interface desktop native
  • Analyse comparative
  • Connexion sécurisée BDD

Rôle de l'application desktop

L'application desktop JavaFX est destinée aux gestionnaires de la société IMMOSYNC pour piloter l'ensemble des chantiers et prendre les décisions stratégiques basées sur l'analyse des devis reçus.

Objectifs spécifiques

  • Centraliser la gestion de tous les chantiers en une interface.
  • Permettre l'analyse comparative des devis selon plusieurs critères.
  • Faciliter la prise de décision pour les gestionnaires.
  • Assurer la cohérence des données avec l'application web.

Interface desktop JavaFX avec tableau de bord gestionnaire et outils d'analyse comparative des devis.

Accès et authentification

L'application desktop nécessite une authentification sécurisée :

  • Connexion base de données : accès direct PostgreSQL.
  • Identifiants sécurisés : chiffrement des credentials.
  • Interface de login : écran d'authentification professionnel.
  • Sessions persistantes : mémorisation de la connexion.

Une fois authentifié, le gestionnaire accède au tableau de bord avec synchronisation temps réel des données de l'application web.

Écran d'authentification de l'application desktop JavaFX avec connexion sécurisée à la base de données PostgreSQL.

Fonctionnalités de gestion

L'application desktop offre des fonctionnalités avancées de gestion :

  • Tableau de bord complet : vue d'ensemble de tous les chantiers.
  • Analyse des devis : comparaison selon prix, date, spécialité.
  • Suivi en temps réel : synchronisation avec application web.
  • Génération de rapports : bilans périodiques et statistiques.

Les gestionnaires peuvent ainsi piloter efficacement l'activité et prendre des décisions éclairées basées sur des données objectives.

Module d'analyse comparative des devis avec critères multiples et aide à la décision pour les gestionnaires.

Architecture JavaFX

L'application desktop repose sur une architecture Java robuste :

  • JavaFX : interface moderne et responsive pour desktop.
  • Connexion PostgreSQL : accès direct à la base de données partagée.
  • Architecture MVC : séparation logique métier et présentation.
  • Synchronisation : échanges temps réel avec application web.
Compétences Java mises en avant :
  • Développement d'applications desktop natives.
  • Gestion de base de données avec JDBC.
  • Conception d'interfaces utilisateur avec JavaFX.
  • Architecture client-serveur robuste.

Réalisations techniques

Modules développés
  • Module d'analyse comparative multi-critères des devis.
  • Tableau de bord avec graphiques et statistiques.
  • Système de synchronisation avec base de données centrale.
  • Interface de validation et de décision des chantiers.
  • Génération automatique de rapports PDF.
Points forts du développement Java :
  • Performance optimisée pour traitement rapide des données.
  • Interface utilisateur intuitive et professionnelle.
  • Gestion robuste des erreurs et des exceptions.
  • Scalabilité pour grandes volumes de données.
